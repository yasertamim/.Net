1- Project description:
this project is a webapplication that allows users to chosse at most four different addresses or geographic locations
from a map so that the application gives the shortest path between these locations where it starts from one loction
and visits all other locations and get back to the first one. this application is based on Salesperson problem.

2- User use-cases
- the user would be able to create an account and sign in
- the user would be able to save a path
- the user would be able to rempve a path from history
- the user would be able to choose locatons directly from the map
- the user would be able to insert the locations to the application as a text input
